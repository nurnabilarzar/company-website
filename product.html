TuxGeo+Ⓡ Our Pride. Your Joy.

TuxGeo+Ⓡ is a Geo Web platform that unlocks the value of your business data by analysing and enriching them with meaningful location related information.

It is a suite of services comprising of four main products:
TuxGeo+Ⓡ Maps for Enterprise
TuxGeo+Ⓡ Address
TuxGeo+Ⓡ Vision
TuxGeo+Ⓡ Transit

TuxGeo+Ⓡ Maps for Enterprise
Customised for Your Success

This ready-to-use and deploy platform will provide geo backend services for your business using your own geo data.

Employ it on premises or the cloud, in a hybrid configuration, and even portable if needed.

It works with OSM, HERE, Bing Maps, TM SmartMaps and Google Maps.

TuxGeo+ for Enterprise includes a complete set of services and Application Programming Interfaces (API) needed for:

1. Visualisation
2. Search
3. Routing
4. Analysis

TuxGeo+Ⓡ Maps Address
Location, Location, Location

A suite of address data services, it will work with most existing systems to automatically normalise, enrich, match, and geocode your data.

TuxGeo+ Address can be integrated with different data sources and databases in different formats. It is able to process millions of addresses and tag them to fit your workflow.

It also comes with tools allowing data operators to correct data and provide insights.

TuxGeo+Ⓡ Vision

TuxGeo+ Vision is a suite of services and components that works with location based image data, from data collection, integration, management and analysis to visualization.

It allows users on the field to collect, update and view information and images via their smartphones.

It integrates with other image data collection systems, and data including 360 street view imagery, and LIDAR.

TuxGeo+Ⓡ Maps Transit

TuxGeo+ Transit is a complete Journey Planner application system hosted on TuxGeo+ platform.

It integrates live updated public transit data feeds from a number of different sources, to provide a standard API (GTFS, GTFS- Realtime) and user interface to allow users to suggest the best transit routes across different modes and operator, e.g. buses, trains, light rails.
